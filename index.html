    Ultimate Tic-Tac-Toe

    React JavaScript CSS3 Spring Boot Google Cloud
    Overview:
    The Ultimate Tic Tac Toe project is an enhanced version of the traditional game, expanding it into 9 big grids with 9 small grids each (total of 81 cells). It introduces player accounts with sign-up and log-in, while delivering a more complex and strategic experience compared to the classic version.

    Key Features:
    • Player Accounts: Players can sign up (generating a unique ID) or log in with their existing ID.
    • Game Mechanics: Classic X vs O gameplay, but expanded to 9×9 nested grids for deeper strategy.
    • Victory Conditions: Winners are determined by capturing large grids through diagonal, vertical, or horizontal alignments. If no clear winner emerges, the player with more large grids wins; ties are also supported.
    • Deployment: Hosted on Google Cloud for online play

    Technical Implementation:
    • Backend: Java Spring Boot (MVC architecture)
    • Frontend (View): HTML, CSS, JavaScript
    • Deployment: Google Cloud Run

    Hospital Management System

    Flutter Node.js PostgreSQL AI/ML Express.js JWT Auth Render
    Overview:
    The Hospital Management System is a full-stack platform that integrates AI-driven healthcare recommendations with modern mobile and web interfaces. It enables patients, doctors, and administrators to collaborate seamlessly, ensuring efficient healthcare workflows supported by AI and doctor validation.

    Core Features:
    • Patient Mobile App: View lab data with doctor-approved recommendations and manage appointments.
    • Admin Dashboard: Upload patient lab data (manual or bulk), manage doctors, departments, and appointments.
    • Doctor Dashboard: Review lab results with AI-generated recommendations, refine or override them, and approve final recommendations for patients.

    Technical Implementation:
    • Backend: Node.js (API services) + Python (rule-based algorithm)
    • AI: Google Gemini (processes lab results and generates recommendations)
    • Mobile App: Flutter (Clean Architecture + BLoC for state management)
    • Web Frontend: ReactJS (Admin & Doctor dashboards)
    • Database: PostgreSQL (hosted on Render)

    Project Impact:
    Developed as part of a team project, this system demonstrates the application of technology in healthcare to improve patient care and operational efficiency. The AI recommendation system achieved 85% accuracy in preliminary testing.
